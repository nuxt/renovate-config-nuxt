# Renovate Nuxt Configs

Nuxt.js presets for [Renovate](https://github.com/singapore/renovate) tool.

## Usage

Add this into `renovate.json`:

```json
{
  "extends": [
    "github>nuxt/renovate-config-nuxt"
  ]
}
```

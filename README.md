# Renovate Nuxt Configs

Nuxt preset for [Renovate](https://github.com/renovatebot/renovate).

## Usage

Add this into `renovate.json`:

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": [
    "github>nuxt/renovate-config-nuxt"
  ]
}
```
